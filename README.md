# Remote Tours Website

A static tourism website created for **Remote Tours Pty Ltd.**, Australia.

## Summary

This project is a responsive multi-page website designed to showcase Remote Tours' experiences, destinations, and travel information for customers exploring Central Australia and surrounding communities.

The website is built using simple static web technologies and can be hosted easily using services such as GitHub Pages, Netlify, or Vercel.

## Features

- Responsive multi-page design
- Tour information and pricing pages
- Community and cultural information
- FAQ and booking terms sections
- Lightweight static website (no backend required)

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Pages

- Homepage
- About Us
- Tours
- Community
- FAQ
- Booking Terms and Conditions
- Pricing
